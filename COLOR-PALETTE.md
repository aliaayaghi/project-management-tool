# Color Palette

Use this file as the source of truth for the project management UI colors.

## Light Mode

### Backgrounds

| Purpose | Color |
| --- | --- |
| Board background | `#F0EBE3` |
| Column background | `#E8E2D9` |
| Card background | `#FFFFFF` |
| Card border | `#D8D0C4` |

### Column States

| Column | Header text | Tag fill |
| --- | --- | --- |
| To do | `#2E5070` | `#DAEAF5` |
| In progress | `#8A4030` | `#F5E0D8` |
| Done | `#3D5C28` | `#DFF0D4` |
